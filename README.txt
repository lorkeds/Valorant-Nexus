Descrição
Sistema em HTML, CSS e JavaScript que usa a Valorant API para exibir informações de agentes, armas e salvar comentários no navegador.

Funcionalidades
• Buscar agentes (nome, descrição, função, habilidades).
• Buscar armas (imagem, categoria, stats e dano).
• Sistema de comentários com LocalStorage.
• Home com imagens e sinopse.

Tecnologias
HTML5, CSS3, JavaScript, Bootstrap 5, Valorant API.

Como usar

Abra o arquivo index.html no navegador.

Navegue pelo menu: Home, Agentes, Armas, Comentários.

Em Agentes/Armas, digite o nome e clique em Buscar.

Em Comentários, escreva e publique; fica salvo no LocalStorage.

API
Agentes: /v1/agents
Armas: /v1/weapons
Parâmetro utilizado: language=pt-BR
